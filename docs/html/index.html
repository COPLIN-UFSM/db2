documentação coplin-db2¶

Bem-vindo a página de documentação do pacote coplin-db2!

Este pacote foi desenvolvido na Coordenadoria de Planejamento Informacional (COPLIN) da UFSM, subordinada a Pró-reitoria de planejamento, e visa atender a necessidade de realizar operações comuns de pesquisa, inserção, remoção e atualização de tuplas em um banco de dados relacional do tipo IBM DB2.

Este pacote é um wrapper sobre o pacote ibm_db, estendendo algumas funcionalidades como permitir o uso de um arquivo json para armazenar as credenciais de acesso ao banco de dados.
